基礎詞彙

本章節旨在將⽇常詞彙與編瞎話詞彙進行關聯對應。
通過逐一對比，加深印象，做到過⽬不忘，歷久彌新。
每個詞彙都安排了例句賞析，以錘鍊編瞎話時的發散性思維。

動機性行爲=吃 / 喝 / 拉 / 啪...

例：吃喝拉啪令人快樂。

——在一些動機性行爲中，動物的獎賞系統相關神經環路顯著地參與了其正性情緒的產生。

負性情緒=悲傷 / 不爽

例：在聚會上不被關注的話，女生會覺得不爽。

——對於羣體活動中無法喚起異性內在注意力的雌性個體，現實與期待的不匹配將導致其負性情緒的產生。

閾下刺激=沒感覺

例：有個男的在 QQ 裏向女神告白，可是女神對他沒感覺。

內隱注意=悄悄注意 / 暗中觀察

例：她好像很認真地聽着閨蜜傾述，卻悄悄關注着那個強顏歡笑的男生。

——外顯感受器雖然呈現出對一方的高度注意，她卻通過內隱注意的方式，觀察着另一個通過高級中樞抑制負性情緒並控制其臉部肌肉傳遞正性情緒信息的雄性生物。

順行性遺忘=斷片

——過量的酒精攝入使個體產生了嚴重的順行性遺忘，在睡眠覺醒之後出現了記憶提取困難的現象，並伴有肛腸末端痛覺感受器的高度興奮。

腦機接口=耳朵

——將兩個發音單元與人類的腦機接口進行適當的接合，一定規律的聲波振動激活了大腦的獎賞系統相關環路。

進階語法

本章節旨將在詞彙的基礎進一步拓展，由單個詞彙延伸到詞組、短語和完整⻓句。

同時通過加強語法結構的內容和思路，以達到隨時隨地編瞎話的學習目標。

在遣詞造句時應假想出⾃己優美而精緻的儀容爲參照，默讀時舌尖輕挑爲佳。

1. 創造並不存在的詞彙

凡是在名詞、動詞、或形容詞後可添加“中樞”兩字的，⼀律添加。

凡是在名詞、動詞、或形容詞後可添加“-- 能神經元”的，⼀律添加。

對於不想做的事物，在名詞、動詞、或形容詞後添加“失用症”。

例：

愛情中樞=愛情 + 中樞

攝食中樞=攝食 + 中樞

性感中樞=性感 + 中樞

多巴胺能神經元=多巴胺 + 能神經元

太陽能神經元=太陽 + 能神經元

語言失用症=語言 + 失用症

起牀失用症=起牀 + 失用症

愛情失用症=愛情 + 失用症

上班失用症=上班 + 失用症

2. 挑大詞

什麼詞都往大了說。

有抽象的絕不選具體的，有模糊的絕不選明確的，有複雜的絕不選簡明的。

儘量選生僻、抽象、詞義涵蓋內容廣的詞彙。

把有關無關的詞彙組成超長詞組。

例：

我們不一樣=由於P 值<0.05，我們在統計學意義上存在顯著差異。

我愛你=你知道嗎，只要你小心地冠狀橫向切開我的大腦，我想對你說的一切，都在丘腦那裏。

好辣=作爲一種非味覺感受，這類辛辣的化學物質激活了口腔皮膚表面負責感受痛覺和熱覺刺激的的TRPA1 受體，產生了一種既痛又熱的奇妙感受。

剛轉身就忘了= 由於在空間延遲反應任務中前額葉皮層對工作記憶的儲存能力與時間存在負相關關係，這個轉身導致了某種程度的記憶損失。

3.記住科學家們的固定開頭

即“雖然。。。，但其中的神經環路機制我們仍然不清楚”。

例：我不知道自己是怎麼愛上他的。

——雖然我們人類對愛情這一現象有了大量的觀察與實踐，很多研究也表明愛情的發生存在於大腦之中，多巴胺、催產素等激素的釋放在愛情中具有關鍵的作用，伏隔核等大腦區域可能參與了愛情的形成，但其中的具體神經環路機制我仍然不清楚。
