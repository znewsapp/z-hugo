同樣由卡西·阿弗萊克主演的兩部影片《海邊的曼徹斯特》和《鬼魅浮生》，剛好可以看作是同一個問題的兩面：究竟，在一場親密關係裏，死者和倖存者哪個更悲哀？

在《海曼》裏，卡西·阿弗萊克扮演了一個失去三個女兒的父親。

他是那個活下來的人，所有的傷痛，都要他來揹負。

他一個人搬到波士頓，做着勤雜工的工作，沉默地度日，可是傷口卻又一次次將它拉回那個叫作曼徹斯特的海邊小鎮。

那裏有溫暖的陽光，也有冷冽的寒冬。溫暖是氣候，寒冬是心境。

這個中年男人，註定無法逃脫負罪，只得生無可戀地活着，這是「倖存者」的悲哀。

而在《鬼魅浮生》裏，卡西飾演了一個披着牀單的鬼魂 C。

他和 M 是一對情侶，住在一座僻靜的房子裏。不想，一場突如其來的車禍，奪去了 C 的生命。

在《海曼》裏有一處情節，哥哥去世，卡西去見他最後一面，他只是注視了片刻，茫然無措地親吻了哥哥的面頰，便匆匆離開。

《鬼魅浮生》也有類似的情景，只不過死的人是 C，前去看他的人是 M。

同樣，M 也感到茫然。他看見 C 躺在病牀上，全身蒙着白色的牀單，安詳、冰冷，無法分辨是「死亡」還是「睡眠」。

可毫無疑問的是，C 確實死了。被車撞死的，頭上還留着一道疤痕。

而死亡和睡眠真正的區別，是「醒來」的方式。

就在 M 走後，C 突然坐起身來，走下病牀，成了一個披着牀單的鬼魂。

他回到那座僻靜的房子，守候在 M 身邊，開始了一段漫長的等待。

這是《鬼魅浮生》最爲特別的地方。

我們看從前的電影，總是更關注一對情侶中活下來的那個人，看他是如何放下和前行的。

而《鬼魅》關注的，卻是死去的那個人。

它採用「亡魂的視角」，拍出了一部很不一樣的悼亡片。而悼念的對象也並不是死去的人，而是「逝去的愛情」。

這場來不及告別的愛情，對活着的人固然殘忍，對死去的人呢？

其實更殘忍。

因爲，活着的人還有未來；而死去的人，卻只有過去了。

當死亡爲一切畫上終點，C 的生命只剩下單向的回溯。他沒辦法用一個新鮮的未來去掩蓋過去的傷痛，他只能愣愣地漚在過去裏，期待解脫。

該怎麼解脫呢？

不知道。

我們只得看着這個披着牀單的鬼魂，遊蕩在房子裏，時刻守候在 M 身邊。

透過他空洞的雙眼，我們彷彿可以看到他的表情，他的哀傷，他落寞的身體。

這個房子的兩個主人，如今只剩下一個。而他被永遠埋在了這個房子的暗面，不見天日。

那薄薄的牀單隔開的，是兩個世界，更是所有接觸的可能。

他唯一能做的，只有注視，只有陪伴。

影片中有一個長達 5 分鐘的長鏡頭，始終對着女主，看她癱坐在地上，一勺一勺吃掉了面前的巧克力派。

她在不斷切割着食物，不斷送進嘴中，不斷咀嚼，面色清冷。

時間一秒一秒的流逝，除了這個機械的進食動作和勺子觸碰盤子的摩擦聲，再無其他。

悲傷就這麼一寸寸地滲透進來，霸佔了全部的時空。

在這個時空裏，表面上悲傷的是「吃」的人，其實更悲傷的，是站在不遠處「看」的鬼魂。

他看着自己心愛的女人，企圖用食物來消解傷痛，卻徹底失敗；看着她衝進衛生間，將那些無效的食物大口吐出。

他只能看着，卻無能爲力，這纔是徹頭徹尾的悲傷。

應該說，比起《海曼》，這部電影「喪」的程度還要更深。

《海曼》中至少還有一些戲劇化的情節，來淡化悲傷的氛圍。

可《鬼魅》沒有，它的悲傷是從始至終的，一點一點的淤積、氾濫，積小流，成江海。

即使是拍攝這對情侶的相處，甜蜜中也瀰漫着憂愁。

它就像是 C 和 M 相擁而眠時，客廳中突然響起的那一聲不和諧的鋼琴，早早便預示了意外的降臨。

它就像是 C 和 M 關於搬家的爭吵：

M 問 C：你想留在這裏？
C：嗯。
M：爲什麼？
C：親愛的，因爲這裏有我們的「歷史」。

是的，C 說的不是「回憶」，而是「歷史」，而這個詞語的含義，我們要到影片結束才能真正明白。

這座房子，對於 C 和 M 來說，是完全不同的存在。

對於 M，它只是個總有異響的鬼屋，是個早早就想搬離的地方；而對於 C，它卻是一個留戀的對象，是一種不願放手的執念。

後來我們知道，其實 C 在這座房子裏，已經等了很久很久。

自 M 離開的那天，C 就站在窗前，目送她的離開。他看車子行駛在暖人的夕陽裏，一路遠去。而房子裏，越發清冷、闃寂，比墳墓還要幽暗。

M 還會回來嗎？

不知道。

但是 C，願意等。

房子相繼有新的房客入住，有單親母親和她的一雙子女，有一羣年輕人在這裏徹夜狂歡，有一個禿頂男人在滔滔不絕地談論着宇宙的永恆真理：所有記憶、存在、不朽，都是妄念，一切終將毀滅。

對於這些阻止 M 回來的人羣和思想，C 動用鬼魂的靈異力量，將他們通通趕了出去。

最終，房子破敗了，再沒有人願意居住。推土機不期而至，將它碾成碎片，化作了塵埃。

C 的等待，似乎也到了盡頭。

在廢墟中，C 看見了隔壁房間那個披着繡花被單的鬼魂，她的等待要更爲漫長，以至於忘了等的人是誰。

此刻，在廢墟上，她終於動了念頭：“我覺得他們不會回來了！”之後便魂飛魄散，被單掉落在地，癱軟作一團。

如果說「死亡」只是「身死」，那麼這一刻的被單掉落，則是「心死」。

“譁”地一下，所有眷戀、期待、希望，全部消失了。

而這，也是 C 的結局嗎？

房子倒塌後，廢墟上又蓋起了新的大樓。

然而，C 並沒有魂飛魄散，因爲他內心期待重逢的火，還沒有熄滅。以至於當他從施工的高地一躍而下時，命運也並沒有終結他的等待，而是給了他一種全新的等待方式。

就像前文寫道的：死了的人沒有未來，只有過去。

C 竟一下穿越回了幾百年前，回到了這座房子建立的起點，用一個「初見」的方式，去等待「重逢」。

那可真是神來之筆。

而我們也終於明白了「歷史」一詞的含義。C 從生前到死後的經歷，恰恰見證了這座房子從建立到倒塌的全過程，而他和 M 的愛情就嵌在這段歷史裏，成爲一個最動人也最心碎的註腳。

房子的主人一直在換，牀單裏，相擁而眠的人總有不同。可對於 C 來說，這座房子和他已融爲了一體，而他身上披的牀單，則是他和 M 愛情的最好見證。

可是，該來的還是要來，該走的還是要走。

C 的等待，與其說是等待 M 的回來，不如說是等待一個真正可以告別的時刻。

那是什麼呢？

心灰意冷？萬念俱灰？死無可戀？

或許是的。

但我想，故事還有另一層含義，並沒有那麼消極。

記得在片頭的字幕裏，影片引用了伍爾芙短篇小說《鬼屋》的第一句：無論你何時醒來，都有一扇門輕輕關上。

《鬼屋》講了一對死去的戀人，回到曾經的房子裏尋找寶藏，最終找到了他們最珍貴的東西，那便是愛情的痕跡。

《鬼魅浮生》雖然講的並不是同樣的故事，但內核卻是相似的。

M 註定不會回來了。

就像影片開始時的那段對話。

M 說：我每離開一個地方，都會留下一張字條，上面寫着詩句。這樣我每次想回去的時候，都會有回憶在等着我。
C 問：那你回去過嗎？
M 答道：沒有。

離開這座房子的時候，M 也留下了字條。那意味着，她也不會回來了，這和之前的每次離開並沒有什麼不同。

而這張字條，也成爲了本片最大的懸念。

上面究竟寫着什麼，使得 C 在看後，瞬間消失？

你可以有很多猜測。比如：我不會再回來了。比如：你不用再等了。又或者，只是一句不知所云的動人句子。

但我想，應該是《鬼屋》裏的那一句：The Treasure yours.——這是你們的寶藏。

這一句，曾在影片中出現過。它就出現在 M 帶新男友回家，C 憤怒地將書架上的書散落了一地，其中一本書打開的那頁，正是伍爾芙的《鬼屋》，而鏡頭給了這句話一個大大的特寫：The Treasure yours.

這是 C 在提醒 M，這座房子對於他們的意義。這裏藏着他們的寶藏——那心中永恆的愛的光輝。

而最終，M 也把這句話留在了字條上，他用這種方式迴應了 C，告訴他：是的，這是我們的寶藏，我只有離開，才能完整地留下它。

那一刻，C 也明白了這句用來提醒 M 的話，其實也是說給自己的。

生死兩隔，留下的是一段完整的、值得一生回味的愛情。有它在就好，人和房子都不是「回憶」必須要依託的東西。

那就走吧。

自始至終，C 等待的都不是 M 的回來，而只是 M 的一句：這段記憶，對我也同等重要。它不是一個說搬走就能搬走的房子，而是一段漂浮在人生裏的彩虹斑駁，永遠無法抹去。

這就夠了。

“無論你何時醒來，都有一扇門輕輕關上。”

可是，你不必擔心。

那不是什麼鬼魂，而只是愛你的人，來看你了。
