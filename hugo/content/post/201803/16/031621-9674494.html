如何評價電影《Annihilation》（《湮滅》）？

“...午夜陽光下的黑水中果實將成熟而黑暗中的金色果實將豁裂揭示出泥土中致命的柔軟...”

——索爾

2014 年，“遺落的南境”第一部《湮滅》出版，並獲得星雲獎，成爲新怪譚的代表作；

2018 年，據此改編的電影上映。

電影的第一個鏡頭，是耀眼的流星尾跡劃過，擊中燈塔。從這一刻起，整個故事的真相被揭穿，也標誌着電影和小說的敘事節奏產生了巨大的分歧。在原著小說中，神祕碎片擊中的並非燈塔，而是陰差陽錯之間被鑄造進燈塔的透鏡玻璃中。而這一關鍵信息，直到第三部，才抽絲剝繭，向讀者隱晦道出。

接下來電影引入了女主角 Lena，一個細胞生物學家。當看到 Trailer 的時候，我便一直在想，娜塔莉·波特曼是個多麼適合飾演幽靈鳥的選擇——原著中生物學家被稱爲幽靈鳥，她的研究領域是生態系統，她如此的關注於此，以至於對家庭和外界一切冷漠以對。也就是說 Lena 同丈夫 Kane 的關係，非但不親密，反而非常疏遠，“我愛他，但不需要他，我覺得很正常”。幽靈鳥的內心，就如同南境的 X 區域一樣，封閉而無法觸摸。而改變這一切的，恰恰是 Kane 作爲第十一期勘探隊成員返回的死，徹底打開了她內心的邊界。

電影由此引出 Kane 的失蹤。這個時候，Kane 邁着怪異而僵硬的步伐，走上樓梯，正如原著所說，“像一具空殼”。兩個人擁抱，接吻，全程 Kane 都帶着無動於衷的被動。他們坐在客廳，桌子上裝着威士忌的酒杯成爲了放大鏡，清楚的表現着 Lena 的手指撫摸着 Kane 僵直的手指——由此貢獻了全片我最喜歡的鏡頭之一。當 Lane 抽離手指的一瞬間，她已經明白了丈夫不再是他自己了。

血跡緩緩的氤氳開來，旋即 Kane 在急救車上被南境局截下。在原著中，Kane 所患的是無法手術的全身性癌變，並死於六個月後，幽靈鳥進入 X 區域之前。影片所沒有表現的是，不僅僅是 Kane，其餘的勘探隊成員也都返回了家中，並且也都死於癌變。至於癌變的真相，則直到第二部才揭曉。

於是 Lena 也被羈押於南境局中，這是故事的二號女主角心理學家 Dr.Ventress 登場。電影自始至終，沒有提及的是，心理學家既不是一個普通的勘探成員，也不是第一次進入 X 區域。心理學家的另一重身份，是南境局的現任局長；而早在第十一期勘探隊出發之前，她就和另一名南境局科學家暗中潛入了一次 X 區域。她之所以對 X 區域如此感興趣，是因爲她就是 X 區域的原住民。當 X 區域爆發時，她還是一個小女孩，而她的好友，燈塔管理員索爾，最終成爲了引發 X 區域的引子。在這之後，她始終心心念念兩人告別時，索爾的那一句“不要忘記我！好好照顧自己！”。

故事接下來的發展順理成章，生物學家得知了第十二期勘探的機會，於是要求前往。第十二期勘探隊員全部爲女性，而第十一期勘探隊員則全部爲男性。這是南境局有意進行的“控制變量法”，以期找到 X 區域反應的模式。

電影大幅省略了勘探隊進入 X 區域的過程，五個人似乎是閒庭漫步之前，就從南境局走入了 X 區域。在原著中，勘探隊都從南境局，進入軍方控制的戒嚴區，再通過一個特殊的開口，方纔能進入 X 區域。也就是說電影忽略了一個很重要的信息，X 區域絕對不是可以隨意進出，如同巨型肥皂泡一樣的存在。“邊界往內陸擴展至距離燈塔約七十英里處，沿着海岸東西兩側各有約四十英里寬，空中直達平流層的下方，地底則到軟流圈上方”而唯一的“入口十二英尺寬，十二英尺高。”而在邊界被出現的瞬間，所有接近或觸線的東西都神祕消失了。而邊界本身是無法跨越的存在。

當勘探隊跨國邊界，Lena 發現自己失憶了——原著中格外強調了 X 區域時間流速的不定和對人心理的巨大影響。勘探隊最終採用的催眠來幫助隊員適應 X 區域。

隨着勘探隊展開勘測，電影的敘事幾乎進入了完全的原創，除了隊員逐一死去。他們在體育場看到了第十二期隊員死前的掙扎和死後的慘狀——看到這裏，我不由得懷疑導演深受 The Last of US 的影響，喜歡大家都變成蘑菇。而原著則沉默和深邃得多，除了大段大段環境描寫，幾乎找不到第十二期隊員的蹤跡。他們就是沒了，消失在了 X 區域中，伴隨着越來越深重的疑問，隱沒在黑暗中。

電影這裏的處理，一如傳統的 B 級片，隊員被猛獸拖走，咬死；化爲人形植物；噴射出孢子，化爲光暈，直到剩下惶然無措的女主 Lena。不多的亮點，是隊員發現女主隱瞞她和 Kane 夫妻關係的氣憤；是猛獸竟能發出受害者最後慘叫的詭異；是枝葉漸漸長出的無可奈何。小說則在沉默中，將勘探隊帶到了“地下塔”中，並驟然加快了節奏。

地下塔的缺失，在我看來是電影劇情改編的巨大敗筆。原著中的地下塔，是燈塔的一體兩面。由燈塔管理員索爾化成的怪物，正是在地下塔的螺旋階梯中，日夜不休的反覆抄寫着聖經，“...午夜陽光下的黑水中果實將成熟而黑暗中的金色果實將豁裂揭示出泥土中致命的柔軟...”；而地面上的燈塔，則是整個故事的開端，神祕碎片的藏身之處，索爾生前的工作地點，也是小說第一部結尾，幽靈鳥發現部分真相的地方。兩座塔，成爲三部曲不可撼動的意象。正如“水井”之於《星之彩》，“高山”之於《瘋狂山脈》，克蘇魯神話的代表手法之一，就是反覆強化一個意象，直至其扭曲並具有強烈的恐怖含義。

原著中，幽靈鳥收到地下塔中神祕孢子的感染，逐漸被 X 區域所同化，感到體內越來越強的光亮感。這種光亮感，正是電影中多次出現光暈特寫鏡頭的原因——不論是片頭的流星還是心理學家化成的光團，光亮感反覆出現，成爲 X 區域的又一個符號。而在幽靈鳥被感染之後，整個勘探隊同樣分崩離析，大家彼此殺戮。

電影推進至此，終於圖窮匕首見——真正的 Kane 早已拉環自焚，而他所託的正是自己的複製人。而小說所由此推導出的情節，則更爲駭人：所以進入 X 區域的人，都不可能返回。心理學家，或者說局長，第一次進入 X 區域時，身患癌症——X 區域無法辨識這一新特徵，盲目複製的結果就是第十一期勘探隊成員的複製人全部患有癌症；前任南境局局長洛瑞，第一期勘探隊的唯一倖存者，也是複製人，並且他之所以能返回的原因就在於他實際是 X 區域派出的間諜；心理學家，也正是和洛瑞彼此利用，推動着一次又一次的勘探。

在鋪墊了整整一百分鐘後，全片最精彩的部分終於呈現。這裏的配樂不得不提，如同沉重傢俱拖動的絃樂，完美烘托出詭異莫測的氣氛。Lena 也終於明白，那個青銅色的古怪人形，不是別的，正是自己的複製人。隨着第二枚燃燒彈的拉環脫出，整個燈塔和片頭遙相呼應，再次發出燦然的白光。

鏡頭一轉，南境局的詢問人員脫口而出，“外星人的目的是什麼？”——這是電影劇情改編的第二大敗筆。前文說到，原著中神祕碎片藏於透鏡之中，又在隨後感染了索爾，索爾發生變異，異化引發了 X 區域；而上述謎底直到小說第三部方纔揭曉。而小說第一部中，模模糊糊的猜測，南境局對於真相的掩蓋，地下塔中索爾大段大段書寫的經文，反而讓小說更像是一部克蘇魯奇幻小說。換言之，正如小說中釋放神祕碎片的幕後組織“科學降神會”的名字一樣，小說是科幻奇幻糅合的風格，既有科學也有降神。電影反而自毀樓閣，把故事變成了一個純粹的科幻故事，甚至是庸俗的外星人侵略故事。

最後電影進入尾聲，兩人擁抱，四目交錯，雙瞳閃爍的微光暗示兩人都是複製人。原著的結尾則更讓人回味，依我之見，湮滅更是一部愛情小說——幽靈鳥，最終打開了內心世界，她無比清楚的意識到自己對丈夫的愛，即便是在丈夫失蹤之後；又或者說，正是這份失去後才懂得的情感，讓小說更加的感人。

評分：7.5/10

劇情：6/10

音樂：9/10

畫面：8/10

娜塔莉·波特曼很美
