你做過最浪漫的事是什麼？

高二的時候，剛剛和喜歡了很久的人在一起，他是農曆正月二十九的生日，前一個月開始就在想該給他準備點什麼禮物。

一天看到學弟在彈鋼琴覺得好像還挺棒的誒！就讓學弟每天晚上放學都教我彈一個小時的琴，那時候總是騙對方說放學有事要趕快回去，還惹得他挺不高興的。

到了生日那天，裝作什麼事都沒有，陪他去買了蛋糕就說是生日禮物了。高中我是一個學生社團的社長，於是打了一份學校千人報告廳的使用申請交到團委去了，說要排練節目拍片子什麼的。

晚上放學陪他和同學們分完蛋糕就假裝回班上自習，其實跑到千報去佈置場地了，把大三角鋼琴推到舞臺中間，和燈光老師把射燈打好就把老師們想法子支走了，然後拿出藏好的蠟燭在臺子上開始擺「XX，生日快樂」，現在想想簡直是作大死，被老師發現我在千報玩火肯定很嚴重。

眼看第一節晚自習都要下課了還沒擺好，就讓來幫忙的同學繼續擺，我去他們班找他，等擺好了給我發個短信。

那天還下着特別大的雨，我跑到他們班門口把他叫出來讓他陪我出去走走

「下這麼大的雨你走啥？」

「走走嘛，雨中散步也很有情調啊。」

於是我倆就打着傘在操場上一圈一圈的走，我因爲內心的緊張話都說不利索，有一搭沒一搭的說着，甚至還問他媽媽最近怎麼樣，心裏想着這倆人怎麼還沒好啊。

終於收到了短信「好了，來吧。」

然後我和他說「寶貝你閉上眼睛，我帶你去個地方。」

想象一下這個場景，他拿着傘我抱着他，往千人報告廳走去。

一步步上樓梯的時候他說：

「這千報吧？」

「你咋知道！！！」

「我又不傻……」

「那你還是閉着。」

我把他放在一千人觀衆席的最中間一排的最中間的位置上

「我喊你睜眼之前不準睜哦。」

然後飛快的跑下臺階，一個箭步竄上舞臺在鋼琴前坐下

「可以睜開眼睛了！！！」

那一瞬間，我覺得自己是理查德克萊德曼，是瓦拉迪斯勞馬克西姆席皮爾曼，彈了一曲特別蹩腳又炫技的改編版生日快樂歌。

「十五歲生日快樂！！！」

彈完之後我對着淹沒在黑色觀衆席裏的他大喊一聲，又立刻跑了下去，他直直的看着我，眼淚留個不停，抱了我好久好久。

最後，我們收拾那堆蠟燭收拾了兩個小時。

幾年前，和男朋友剛在一起兩三個月的時候，他生日。

我沒什麼錢嘛，很窮很窮，就爲他寫了一首歌，把伴奏藏在他筆記本里 C 盤第一個文件夾一直點下去的最後那個文件夾裏。

然後我把歌詞寫在一張信紙上，交給他的室友，囑咐室友一定要零點準時交給他。

零點的時候他一頭霧水地同時接到我的電話和室友交給他的信紙，在我的遠程指導下點開了 C 盤裏第一個文件夾裏的第一個文件夾裏的第一個文件夾......裏的伴奏。

他就讀着歌詞、伴着電腦播出的伴奏、聽着電話裏我唱完了一整首歌。

那些歌詞都是我們一起經歷過的事情，他當時感動壞了！

那個時候他還在追求我，各種膽大心細不要臉，不喜歡不反感，但當時我在上海，他在北京，我直接沒考慮過他，更多的是看不上他，在衆多追求者中，他實在太不起眼，朋友都佩服他的勇氣。

有一天，他到了北京八達嶺長城在好漢坡，給我打了個電話興奮的說：我喜歡你，就和這長城的城牆一樣堅固，和這長城的磚頭一樣踏實，以前不明白爲什麼長城可以活這麼久，今天我明白了，我沒辦法幫你修一座長城，但秦始皇讓長城能留到今日就是爲了讓它來承載我對你的愛吧。 我回答說：我跟你不熟。當時他就直接掛了電話。一個月後，收到他從烏鎮寄過來的明信片：“我在烏鎮郵局的公共牆壁上，給你留了一張明信片，上面有一句你不喜歡的話。一個多月後烏鎮景區龍形田裏的薰衣草就都開了，如果這張明信片能等到那個時候繁花的孤獨，希望能夠迎來你紫色的親睞。“ 不知道爲什麼，我心裏沒怎麼想着他，但會一直惦記着那張明信片和這句話，還在想那句我不喜歡的話是什麼。

三個星期後的週末，我和姐妹去烏鎮玩，在烏鎮郵局內的紀念牆上，我很快就找到了那張明信片。

文字內容是：霧起時 / 你就在我懷裏 / 這烏鎮充滿了溼潤的芳香 / 霧散後 / 卻已是一生 / 鎮空水靜 / 只剩下那在千人萬人中 / 也絕不會錯認的背影。 天吶，在熙熙攘攘的人羣中，在衆多的明信片中，我看到在牆壁上歪歪斜斜訂着的這張明信片，三行漂亮的行楷，我真的深深的震撼了，眼睛溼潤了。

他知道我喜歡去烏鎮，他知道我喜歡薰衣草。他應諾給了烏鎮郵局的衛生員了 521 塊錢爲他看着那張明信片 2 個月內不被覆蓋和掉落。他是搞計算機的工科男，那個時候他的公司非常艱難，他壓力非常大。

2015 年復活節前後，我和男票來到了西班牙的 Ibiza 小島度假，那是我倆第一次單獨長途旅行。

這個島是歐洲著名的 party island，標準的夜生活就是喝酒、跳舞、享受音樂和海鮮，然後和另一半共度良宵——

是夜，涼風習習，海浪聲聲，月明星稀，遠處有隱隱約約的 blue jazz 傳來，氣氛真是美妙。

我和男票搬了兩把椅子，坐在酒店房間的陽臺上，看着月色下的沙灘，吹着小風喝着小酒，開始聊一些各自小時候的故事。

聊着聊着，驚喜地發現我倆都非常愛好文學，以前也都寫過一些散文、小說，發表在了各自的博客裏。

於是我說：“我想看你的文章。”

然後，男票就登陸了好久不用的博客，選了一篇他寫的散文，開始用非常溫柔的聲線——讀！給！我！聽！

我記得那天，房間裏檯燈的光線有些昏暗，就着陽臺上的月光，男票半邊臉的輪廓隱在窗簾後的陰影裏——他念給我的那篇散文，講述的是自己以前在山區支教時候的故事：而那時的他，是幾十個山區孩子的語文老師。

有那麼一瞬間，我彷彿回到了小學的語文課堂，發呆一樣看着文質彬彬的“老師”，用抑揚頓挫的語調讀着美文，思緒也跟着他的文章，從西班牙的小島，飛到了萬里之外的大山裏……

讀罷一篇，他笑着說：“該你了。”

於是我也翻出了博客裏很久以前的文章，開始跟他朗讀起了自己的故事。

就這樣，你讀一篇，我讀一篇，我倆都忘記了時間的流逝……

等我倆都發覺有些睏意時，一擡頭，正看到朝陽從海平面上緩緩升起。

 

直到現在，我仍然時不時想起那個浪漫的夜晚——異國他鄉孤男寡女共處一室，有酒精、有音樂，四處都是尋歡作樂的人們……然而，我們只是無慾無求地朗讀着彼此的散文，整整一夜，直到天光泛白。

愛情的美妙之處大概就是一個矯情的文青找到了另一個矯情的精神伴侶，從那天起，我開始相信電影《朗讀者》裏的某些情節是可以真實存在的。

沒錯，我們就是那種至今還會用鋼筆互相寫信的人。（圖爲男票朋友圈）

有什麼事情是你知道了卻假裝不知道，看到了卻假裝沒看到的？

背景：我和我媳婦，各自都是對方的初戀，相戀六年，結婚一年。

最近想買個藍牙耳機，在網上選來選去看中了蘋果的 AirPods，我正好用的蘋果手機，簡直是絕配。正準備下單的時候，忽然想到 28 號就是七夕了，還沒給媳婦準備禮物。於是在網上找禮物，看中了一條項鍊，兩千多。這樣一來就陷入矛盾之中，如果買了項鍊，就沒法買 AirPods 了（工資不高，見笑）。後來一咬牙，媳婦重要，下單了項鍊。耳機等下次發工資了再買吧。

重點在後面。

昨天項鍊到貨了，看了看還算滿意。離七夕還有一段時間，我得先把項鍊藏起來，到時候再拿出來給媳婦。於是在家裏找藏東西的地方。家裏有一個櫃子是專門放冬天的厚被子的，我翻了翻看看藏在那裏面行不行，結果翻到了一個盒子，打開之後你猜是什麼？全新的 AirPods！！！盒子裏還有一張紙條：老公情人節快樂！我當時那個心情啊，媽的我上輩子是拯救了地球嗎？冷靜下來後，回想起來前幾天，我在網上選耳機的時候，媳婦就在旁邊，還問我哪個好，我就隨口說了句我喜歡蘋果的但是有點貴，原來她都記在了心裏，還給我準備了這個驚喜。然後我默默的把耳機放在原處，找個其他地方把項鍊放了起來。

我當然不會告訴媳婦我已經發現了她的祕密，我只會靜靜地等到七夕節那天，等她拿出禮物，看到我一臉的驚訝。

對了，媳婦是在網上買的耳機（別問我怎麼知道的，盒子裏有發票），1488，官網才 1288，這個敗家媳婦兒！
