前言：經歷過一系列聲淚俱下的抗爭歷史後，我覺得有必要來梳理一下關於“男 / 女朋友爲什麼生氣了”和“男 / 女朋友爲什麼又生氣了”的原因，萬一碰上了一個不玩遊戲甚至討厭遊戲的“真愛”，或許能幫到你一點。（找什麼男 / 女朋友！？是遊戲不好玩還是番不好看？）

其實說男女朋友，更想聊的是泛指的一個羣體：我們身邊那些不喜歡我們玩遊戲的人，這些人可能是家人，朋友，同事。但是我一個人在外省工作，朋友都是玩遊戲的主兒，同事下班之後沒有交集，所以就只剩女朋友以及朋友的男 / 女朋友這一類研究樣本了。也許我的探究過程能幫助到你，怎麼認知和對待身邊那些不喜歡你玩遊戲的家人 / 朋友 / 同事。

當然了，對於這件事情的認知我是層層遞進了，所以我也按照我認知的過程去寫，大家可以看看到底哪一層更適合你，或者說你還有我沒想到的層次，也歡迎留言交流。

第一層：都是年輕人，他不喜歡只是還沒玩過遊戲罷了

可能在玩家心裏，對遊戲抱有敵意或者刻板印象的人多存在於我們的上一代人中，爸爸媽媽爺爺奶奶這一輩在這個世紀初經歷了遊戲是洪水猛獸的媒體洗腦後很難不對遊戲抱有成見。但是在我們同輩甚至同齡人中，對遊戲不待見的人可不少。

在一開始，我對於同齡人不喜歡遊戲的人羣認知是，哪有年輕人會不喜歡遊戲的？只不過還沒玩過遊戲罷了，有句話怎麼說來着，遇到自己喜歡的男人之前，男人都以爲自己喜歡女人（劃掉）。

也確實有這樣的羣體存在，對於電子遊戲抱有不屑一顧的態度，但是在接觸之後卻喜歡和沉迷得不要不要的，所以我也抱着試試看的心態，向不喜歡遊戲的女朋友推薦過一些遊戲，或者陪她一起玩。

手柄握姿充分說明了男女差別。。。

理想的情況是，當你推薦給他一些遊戲後，他見識到遊戲的魅力，開始理解你的愛好，兩個人一起玩遊戲其樂融融，豈不快哉！？

emmmm……但是我失敗了，向女朋友安利和演示了不少遊戲，她也玩過一些，但是大多沒有後文，也許女性對於遊戲的喜好有特殊的地方，我也儘量考慮到了女性審美和她的愛好題材，但始終沒能把她帶入遊戲圈子。

後來想想，我們這代人，哪怕自己不玩遊戲，周圍人多多少少都會有遊戲玩家，在企鵝的滲透下，我們的社交圈不可能與遊戲割裂（現在還有人不知道亡者農藥嗎？），在這樣的生存環境中，都沒有喜歡上游戲或者說進入遊戲圈子，不可能是不知道遊戲或者不瞭解遊戲的人；甚至還有可能是嘗試甚至深度體驗後退坑的人。所以他不玩遊戲或者不喜歡你玩遊戲，只是因爲“他沒玩過”的情況在同齡人，或者說年輕人當中並不適用。

但確實還存在着沒有體驗過遊戲和不瞭解遊戲的人。就是我們父母往上這一代。

不知道大家有沒有看過一個“老年人玩 GTAV”的視頻，因爲遊戲題材的原因現在國內各大視頻網站都找不到了……一開始我點開這個視頻的時候的預期是：可能老頭老太太們會感嘆一番遊戲畫面，然後開始批判遊戲題材。怎麼能搶別人車呢！？怎麼能在街頭開槍呢？哇，居然還和警察對抗！？但現實是：

他們搶車玩得很開心並且和路人警察對射得不亦樂乎

可能你會說國內外環境不一樣，外國開放很多。但其實人性是相通的，我小時候家裏人也對我玩遊戲這事兒很反感，但自從經過了 wii，kinect，以及智能手機的洗禮之後，大家晚飯後都心照不宣的捧着手機或者打開遊戲機了。再沒人說什麼一天就知道玩遊戲，玩遊戲傷眼睛之類的話了。因爲嘗試過後他們都知道，這只是一種正常的消遣，和看電影讀書釣魚沒什麼兩樣，而且還挺好玩。

結論：

長輩對於玩遊戲的反感和恐懼來自於對“遊戲”這個事物的不瞭解和未知 。

解決辦法：讓他玩遊戲就好，門檻越低越好，先讓他願意瞭解以及去了解。

第二層：遊戲是生活和事業的墳墓

既然不是因爲對遊戲的刻板成見而導致他對你的遊戲行爲不滿，那還會是什麼呢？我見過不少因爲男 / 女生玩遊戲導致關係破裂分手的故事，其中的固定句式大概是這樣的：

他就天天玩遊戲，工作也不上心 / 課業上掛科不少，回家 / 寢室就是玩遊戲。

這句話的頭尾我們留在後面說，因爲回家就玩遊戲這事情對於玩家來說是很正常的，問題在於中間這句“工作也不上心 / 課業上掛科不少”。

作爲遊戲玩家的我們也不得不承認，在我們之中存在着很多因爲自制力不夠強而導致生活事業一團糟的人。沉迷遊戲其實就像癌細胞，有的人發作，有的人沒發作，但不代表沒發作的人身上就沒有癌細胞。沒有自治力的人，除了遊戲，他還會有別的沉迷事物，古代人玩個小蟲子、鬥蛐蛐兒都能迷到傾家蕩產茶飯不思呢。何況遊戲呢？

廣爲流傳的別讓男人帶孩子組圖之一

所以如果真的是因爲自己過分沉迷遊戲而導致生活一團糟，我覺得你被反對玩遊戲是正確的。這種生活方式是對自己和別人的不負責任而且影響自身健康！，遊戲固然好，但是終歸只是生活的一部分而不是全部，耳邊迴響起來一句話：“多花時間出去運動運動，別老宅在家裏”。

結論：

因爲玩家太過沉迷而導致被反對玩遊戲

解決辦法：你的家人朋友是對的！減少對遊戲的依賴吧！

第三層：除了遊戲，啥都不知道，無趣的人

在確定了前兩層都不符合我的情況後，我開始嘗試和女朋友溝通問題的本質所在，在多次白刀子進紅刀子出的交鋒之後，我得到了這麼一個可能的答案：因爲好像我的娛樂活動只有遊戲，導致和她沒有共同語言，聊遊戲她不懂也沒興趣，她和我聊的話題我也帶不動。

遊戲玩家的生活也可以很精緻而多彩，玩家就註定宅嗎？玩家也有玩家的旅行方式。

參加活動就不說了，各大展會不就是一趟遠程旅行嗎？

不喜歡去人多的地方擠，還可以聖地巡禮啊 ，爬爬山呀，實在不喜歡出門或者經濟實力不夠，可以在家裏看看書呀（太多了我就不放鏈接了），打打拳，砍砍人啊（如果你能因爲這篇《兵擊指南》拿起劍來，我就滿足了！）甚至做碗好吃的面也能滿足他吃貨的心。

或者像衆多機核作者一樣，寫寫文投投稿，讓你的他知道你的生活和愛好並不是那麼乏味，反而異常精彩紛呈。

我想這纔是“在機核你甚至可以……”這個梗的最終奧義，展示玩家多彩的生活，帶給玩家更加多彩的生活方式。和上文說的一樣，真正無趣的人，就算不玩遊戲，相處起來也依然無趣。

誰要說遊戲愛好者的生活都很無趣，我會推薦他看看大狗老師的文章。

非常感激大狗老師展現的這麼一幅玩家衆生相，玩家也是人，並不是說因爲玩遊戲而和其他人有所不同，是人，就有自己的人生，有自己的故事。或悲哀或簡單或樂觀，哪會有無趣的人生呢？

結論：

因爲你只和他聊遊戲導致他對遊戲異常排斥

解決辦法：多逛逛機核等各大網站，解鎖更多生活模式，也多去嘗試瞭解他的愛好，然後來寫一篇[不玩遊戲幹什麼]。或者，不和他聊遊戲本身，而是聊遊戲裏的文化現象和故事。遊戲只是一個載體而已，和聊電影、劇、小說沒兩樣。也可以聊聊那些你看到，知道的玩家故事，玩家只是一個身份，他們的故事和家長裏短的八卦也沒區別。

第四層：玩個破遊戲居然還花那麼多錢！？

這個問題得分兩層來說，一種是，花錢買遊戲 / 周邊，另一種是，花錢購買遊戲服務。

對於愛好的投入，只要在一個正常範圍內，我想都是可以接受的，哪怕抽根菸，哪怕吃個冰淇淋薯條，都要錢。但爲什麼唯獨會有不少人對於在遊戲中投入金錢這件事看得尤爲的不適呢？

在大多數人心中，吃薯條是免費的，有不少人去麥當勞偷來了薯條分發給大家，而現在真正喜歡薯條的人都知道我們得去麥當勞吃，這樣麥當勞纔開得下去，才能繼續給我們炸薯條。所以其實能理解那些看不慣“買遊戲”行爲的人羣，因爲曾經，吃薯條是不要錢的，現在也依然可以免費吃薯條，花錢去吃，不是敗家子嗎？

而除了薯條之外，麥當勞或者是其他和麥當勞合作的廠家，會出一些小玩具，搭在套餐裏面售賣，就會有人想買一個回家作紀念，本質上和去旅遊景點看風景然後帶回幾個紀念品沒什麼兩樣。這是種消費觀念的差異，有人能接受爲了紀念和回憶買單，但有人不能，本來花錢去吃飯的，爲什麼要多花那麼多買個玩具回來？

大概是爲了紀念這頓飯的美好吧。

接下來說說第二種。

有其他商人看到麥當勞賣薯條賺錢了，就自己也開始做薯條，並且免費向所有人提供，以前的免費薯條是偷來的，這個可是堂堂正正的免費！（所以上面說的免費薯條現在太多了，不論是偷來的免費還是真正的免費，搞得人們覺得免費薯條纔是主流老大。）但是！番茄醬，請付費，誰會喜歡幹吃薯條呢？爲了更好的用餐體驗，不少人掏錢買了。還有！座位請付費，站着吃薯條多累呀，於是又有人掏錢買了個座位。最後，你可以多花點錢，讓服務員把你點的薯條送到你的位置上，甚至餵你吃下去，不用點餐後苦苦等待，自己端盤子去找座位，於是又有人掏出了錢包。旁人不解：爲什麼別人吃薯條一分錢不花，你吃薯條像個無底洞似得往裏扔錢啊！？

因爲我比別人吃得更爽啊。

我想遊戲玩家一眼就能看懂我在說什麼，裏面的薯條，麥當勞等等指代什麼。某種程度上，只要在經濟能夠承受的範圍內，我覺得是否在遊戲上消費，消費多少達成共識，更多的是與彼此間的消費觀念有關。

其實這和“遊戲”本身無關，就算愛好不是遊戲也會有這樣的矛盾：

“你看看你，打個球買幾千塊的球鞋和球衣，還花錢去室內籃球場打，我看樓下老大爺穿個綠膠鞋穿個破褲衩在隔壁學校操場也打的很開心啊，爲啥你要花這麼多錢？”

結論：

小部分情況，是對於遊戲免費的誤解，而絕大部分，其實是彼此消費觀念的不符合。有的人可能就是無法接受爲精神愉悅和更好的服務以及體驗買單。

解決辦法：

如果只是對於遊戲有誤解，請把這個麥當勞薯條的故事講給他聽，他應該就能理解了，如果是消費觀念上的差異，擰得過來就擰，擰不過來的話……求同存異吧。

（你們以爲存私房錢是幹嘛的？不就是爲了實現自己那些小夢想嗎？雖然小夢想也有好有壞）

第五層：爲什麼不陪我！？遊戲比我還重要嗎？

我想看到這個標題，不少朋友都頭皮發麻了。不知道這句話紮了多少顆心。當然，也包括我。很多時候，家務也做完，工作也完成，而她也在做自己的事情時，你覺得哎呀歲月靜好啊，打開機器玩玩遊戲吧！

“BOOM！！！！！” 炸了……

當然不是遊戲機，而是你身邊那個人。伴隨這一陣指責：就知道玩遊戲，一有空就玩遊戲，什麼時候不能玩遊戲？爲什麼不來陪我一起看劇／逛淘寶／睡覺／聊天／刷微博？

玩遊戲變成原罪了？

在這種情況下，恐怕是的——不止玩遊戲是原罪，做任何與他無關的事情，都是原罪。我想大家都有體驗過放寒暑假回家一段時間後的老媽吧。“都幾點了！還不睡 / 起牀 / 吃飯！？”以及“成天就知道出去浪 / 蹲家裏”。一種看你做什麼都不順眼的狀態。比扎針的容嬤嬤還可怕。

我以一個理科直男的思維進行百轉千回的思考，多次進行控制變量實驗之後終於發現答案：

事情不重要，情緒纔是關鍵。如果情緒不對，你做什麼事情都是錯的。

看似事件衝突，本質上其實是情緒的對立。相信我，其實遊戲只是個個例，上面說的那種情況還會出現：

“就知道 X!，難道 X 比我還重要嗎？”

X = 看書 / 睡覺 / 寫文 / 畫畫 / 發呆 / 看手機 /（以及任何你減壓時會做的事）

而這時候要解決的，和遊戲無關，而是他的這種“被忽視感和不被服從感”造成的不滿情緒。因爲我們在專注於自己的活動時，往往會忽略身邊的人，陷入自己的小世界之中，特別是遊戲這種娛樂形式；所以遊戲纔會往往成爲萬矢之的。明明這個人就在你身邊，就在你眼皮底下，而他卻對你視而不見專注於其他，這種被忽視的感受產生的情緒是非常巨大的，會被理解爲對他的不耐煩和無視。

而此時我們自己也需要有情緒進行消化，遊戲以及其他愛好對於我們來說，或許是生活的一部分，也是減壓，卸除生活重擔的一個出口。拿我自己來說，明顯能感受到的是，在一定時間的遊戲之後，我會有更高的熱情和效率去面對生活和工作，像是一個加油站。所以眼看就到長途高速公路上的休息區了，卻硬生生被叫回來繼續上路前行，多多少少，得學會面對自己心裏的失落和不快的情緒。

遺憾的是，這一層貌似不是一個人能解決的，需要彼此合力。

結論：

遊戲只是爲你們之間情緒的對抗背了鍋而已。

解決辦法：要麼化解情緒，要麼預防情緒的產生。前者，需要雙方都能體諒對方的想法“他只是想我更關注他”，“他只是想放鬆一下。”時刻記住這兩句話，不要把事情上綱上線。後者，需要雙方給彼此更多的個人空間和時間窗口。當彼此進入對方的生活，不留空間真的會把人壓榨的喘不過氣。不管是父母子女還是男女朋友妻子丈夫。如果真的沒法加入到他的小世界，不妨也給他的小世界留一點空間。

最後說幾句：

其實聊完我所遇到的所有情況，我們會發現一點，其實所有“他不讓我玩遊戲”的核心問題大都不是出在“遊戲”本身上，也不是“他愛不愛你”這個事情上（頭圖和題圖只是在搞事情），讓他正確認識遊戲這種事物之後，更應該關注的是你們生活相處方式的差異，價值觀的差異，情緒的矛盾。而作爲玩家，這些問題自然最可能暴露在“遊戲”上面。所以大家放輕鬆，遊戲沒有錯，錯的地方，永遠在人與人之間。

希望天下的玩家都擁有愉快的遊戲生活！
