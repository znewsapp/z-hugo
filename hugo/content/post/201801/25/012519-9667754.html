CRH4 其實是存在的，它就是現在的南車株機的子廠商廣東南車，但由於總所周知的原因，鄙棄了 4 的廠商代碼而使用了 6，但依然可以從它的車組編號上體現出來。

CRH6A-0401，原車號 CRH6A-4001，如此類推……

3.

CRH7 其實也是存在的，它是日立當初想引進入中國的 200km/h 等級城際動車組的一種。

4.1

普速客車的席別，其實不是隻有你在車廂外表上看的那幾種（硬座硬臥軟座軟臥），在建國初期它們之間可以是互相組合的，例如……

軟座 + 軟臥，則是軟座臥車（RZW）

軟臥 + 軟座，則是軟臥座車（RWZ）

兩種車的區別在於席別比例，下同。

軟臥搭配硬座是不是很突兀？所以不會有【軟臥硬座車】這種奇葩出現的。

軟座 + 硬臥，則是軟座硬臥車（RZYW）

軟臥 + 硬臥，則是軟硬臥車（RYW）

硬座 + 硬臥，則是硬座臥車（YZW）

軟座 + 硬座，則是軟硬座車（RYZ）

4.2

看暈了吧，列車上總要吃飯的，下面加個餐車。

軟座 + 餐車，則是軟座餐車（RZC），這個席別其實沿用到現在，動車上的 ZEC 就來自於它

硬座 + 餐車，則是硬座餐車（YZC）

臥鋪加餐車，開玩嗎，所以沒有！

建國初期沒有單獨餐車，只有廚房車（CF）

而硬座會慘一點，通常廚房車會和它合造，成爲了硬座廚房車（YZF）

4.3

是不是已經很暈？後面還有呢，行李咋辦呢？所以就有了

軟座 + 行李，軟座行李車（RZX，或者是 RZXL），這個種類的客車現在還可以在廣深鐵路上看到。

當然廣深新引進的 25T 行李合造車直接標 RZ，但依然可以從車廂外表看出區別【那大大的一個行李裝卸口】。

硬座 + 行李，硬座行李車（YZX，或者是 YZXL）

硬臥 + 行李，硬臥行李車（YWXL）

軟臥相對高級一些所以就不摻和了。

4.4

對了，那列車郵局咋辦呢？苦逼的鐵道部自然不會忘記它，於是就有了

行李 + 郵政，XU，這種車廂現在能在昆明米軌上看見。

硬席總是很慘的……來個終極合體版吧

硬座 + 行李 + 郵政，YZXU，硬座行李郵政車。

硬臥 + 行李 + 郵政，YWXU，硬臥行李郵政車。

4.5 那些代用車

在建國初期的鐵路上，哪來那麼多客車給乘客坐，鐵路都是貨運爲主的，那麼運力不足怎麼辦呢，這時候代用車就來了。

代用車和貨車的區別其實是在於，它是貨車改的，但改成了客車的樣子，而不是僅僅在貨車裏頭加點兒凳子。

當年的代用車主要有這幾個型號：

代用坐車，ZP

代用行李車，XP（非微軟生產）

代用郵政車，UP

代用軟臥車，WP

零擔車，DP、化肥農藥零擔車，VP（和前面的相比，這個車有人押運）

【零擔是個什麼鬼？早起鐵路人和貨一起運輸的時候，有些不太方便和其他貨物放一起的，就裝在零擔車上，可以理解成快遞的雛形】

這幾種代用車主要是用雜型貨車改造而來，在 90 年代還會有一些支線鐵路在使用。

現在自然是都沒有了……

4.5 那些莫名其妙的特種車（TZ）

其實各位不要以爲看到 TZ 就是大水錶，事實上 70 ~ 80 年代列車上剛開始有空調的時候，那個鍋爐車和發電車，都叫 TZ

甚至到了 25 型和 25A（就是俗稱紅皮車）開始運用的時候，那個掛在列車最尾的空調發電車，型號還是叫 TZ

當然現在爲了各種需要，連個地鐵押運車都能叫 TZ 了……

4.6

那接下來介紹一些可能根本沒人聽說過的特殊型號

醫務車，YI（對，和硬臥車型號直接撞了，所以改成了苦逼的 YI，說白了就是列車醫院，但數量寥寥可數，有些 YI 後期也改成 YW 了）

衛生車，WS（可以理解爲藥房或者病房，和上面醫務車搭配使用）

文教車，WJ（其實就是流動科室）

瞭望車，LW（掛在列車尾部，在今天的意義應該是相當於旅遊觀光一類，其實當年更多用於軍事用途）

公務車，GW（這個不用說了，水錶的一種）

維修車，EX（爲什麼不叫 WX 呢，因爲它又和另一種特種車紅外線檢測車【利用紅外線檢測鐵軌狀態，發現不行就通知工務修復】的型號撞了），但因爲鐵道部從未真正給這類特種車統型，所以一臺維修車，既可以叫 WX 也可以叫 EX，更可以使用下面這種型號……

軌道檢查車，DJ（上面絕對沒有唱片騎師），這種車和 WX 的最大區別是它的檢測裝置位於列車尾部，但同時由於上面提到那點的原因，同樣的一臺車，也可以叫 WX，也可以叫 DJ。

4.7

新上任的鐵總，算是真正終結了一下客車型號的亂象，開始真正給特種車統一定型了，但其實也沒那麼神祕，就是從兩個字母升級成三個字母而已

如果這節車是宿營車的話，那麼它的型號就是

T（特種）+K（客車）+S（宿營）=TKS

如果是合造車則是

T（特種）+K（客車）+H（合造）=TKH

那發電車則會是

T（特種）+K（客車）+F（發電）=TKF

當然真正的大水錶肯定不會這樣標，知道的鐵道飯絕對要悶聲發大財

4.8 試驗車家族

試驗車（SY）和維修車一樣，都是沒有統型的車型，基本上只要是試驗用途，或者只沾一點兒邊的（例如放存放工具）也能標 SY，它的具體用途只能靠車體外部的文字標明，例如一些比較規範的鐵路局會寫上【隧道試驗車】、【接觸網試驗車】等用途。

5. 普通旅客列車機後一位通常不載客這個規則，其實是來源於一次重大事故（楊莊事故，血與淚的教訓）

6. 那些稀奇古怪的貨車車型

A，礦石和砂石運輸車，和現在的礦石車不同是它的裝載範圍大很多，現在只能在一些礦區能看到（外形和敞車相近）

F，通風車，這種車型在日本和臺灣依然有，裝載一些易腐貨物，例如蔬菜。

現在被保溫車（B）取代了

H，活魚車，顧名思義運魚的（但因爲技術不過關，掛得太多，被公路取代了）。

J，家畜車，老一點兒的車迷可能會有影響，因爲一般都會伴隨着濃郁的“香味兒”，現在大多改成汽車運輸車了（J5SQ、J6SQ）

小汽車運輸車型號裏面的 J 是代表產權屬於中鐵特貨，真正型號其實是 SQ。

M，煤車，功能單一自然被敞車取代了。
