情侶性格相似好還是性格互補更好？

也許是太多童話故事結尾只到“王子和公主從此幸福的生活在一起”，並沒有描述他們婚後的生活，所以讓大家產生“找到白馬王子 / 公主=幸福一生一世”的錯覺。很可惜也很慶幸，親密關係不是“押寶”，不是押對人就毫不費力的幸福一輩子，也不是押錯人就整天以淚洗面定終身。如果硬要回答這個問題，那麼回答將是不論你是找到一個性格互補的伴侶還是性格相似的伴侶，都有可能幸福也有可能不幸福，因爲決定你幸福與否和多久的並不是你們的性格相似與否，而是你們會不會交流！

與其糾結哪一種讓你幸福更久，還不如自己創造幸福，因爲幸福是在你自己手上的呀。因此，在親密關係中延續幸福的重點其實不在“尋找”，而是在“交流”！

很多人可能會問，交流這件事每天都在發生，我們很小就開始說話，每天都在大量的說話。這有什麼可講的呢。

不得不說，語言是一個有魔法的東西。一些攻擊性的語言可以馬上讓你的伴侶發瘋。積極美好的交流，伴侶們會迅速變成一個可愛的小王子或者小公主。

當我們在學校的時候，我們學習如何使用一門外語，甚至如果可能，學習跟計算機，外星人對話。但是我們從來沒有學習過，我們該如何跟伴侶溝通。我聽過有人說，我希望我跟我女朋友的溝通就像我跟計算機一樣。可見對有的人來講，交流比 C 語言還有難度。我問他爲什麼你覺得跟計算機交流更容易呢。他說因爲我知道我怎麼能讓電腦在表達什麼，但是我真的不知道怎麼去跟女生溝通。

情侶關係屬於親密關係的一種，親密關係的交流是一個很大的話題，親密關係不單單指情侶，它也指你和你父母或者非常親密的朋友。

我們先從一個例子開始……

例子的主人公就叫小紅和小明好了，小明和小紅已經約會了一段時間了。小明打算五一假期跟朋友們一起聚一下。但是他沒說要邀請小紅一起去，可能還沒來得及說。小紅就開始比較焦慮了，她開始擔心這意味着什麼：
A.是不是小明只把他當成臨時女朋友呢？
B.也許小明覺得帶我出去很丟臉？
C.難道他朋友裏面還有他的前女友和曖昧對象？
D.他根本不想帶我進入他的朋友圈？

小紅跟男生並沒有說清楚這件事，因爲她覺得如果她主動提出來會顯得很丟臉，顯得自己非常渴望對方帶他去參加聚會。相反，她暗示小明說：“五一我有好幾個朋友要找我玩，我還沒有決定去哪個。”實際上，她沒有任何的計劃，但是她不想讓自己看起來她在等待男生的邀請。
小明並沒有注意到她的暗示，女生覺得自己的猜想都是對的。她開始憤怒和生氣，甚至覺得要放棄這段感情。最後忍無可忍女生終於爆發了，她對男生說：“我覺得你根本都不在乎我，也許我們這段感情從一開始就是錯的！”
小明問她爲什麼？她說：“你都不帶我出去見你的朋友！”
男生說：“啊？是這樣啊，我跟高中那些男生一起聚會，怕喝酒吃串什麼的，你會覺得無聊纔沒有邀請你呀。”
也許小紅猜對了，小明真的覺得小紅不怎麼樣不想帶她，也許小明真的是跟同學要去喝酒吃串，交流的好處是我們可以基於事實做判斷，不是基於我們腦中的猜測做判斷。

爲什麼大部分時候 我們都不願意去表達自己？

第一，我們的文化很多時候不鼓勵女性去表達和溝通，在傳統文化裏面，女生要矜持、要含蓄、這樣纔是高雅的。但是在現實生活裏，過度的含蓄會增加雙方交流的難度。

第二，我們從小到大的成長環境。更多的是讓我們去服從學校的規則，家庭的規則，要求我們整齊劃一，我們很少有機會表達我們的感受。80、90 出生的父母們，因爲年代背景的侷限性，很多時候會忽略孩子的感受和表達。所以大家的成長過程沒有培養起來自我表達的習慣。當一個人長期處於一種不被聆聽的狀態，很多時候她潛意識就會覺得我是不重要的，或者我的感受是不重要的。久而久之她會關閉掉表達。

第三，當我們提出自己的需求，這意味着我們有被拒絕的風險。向別人提出要求確實是有風險的，這會讓主動權在別人那裏。當我們被拒絕的時候我們會感覺非常糟糕，我們會覺得我們可能是很糟糕的人，我們會擔心自己被拒絕，這會增加我們提出需求的恐懼和難度。如果我們是這樣想，雖然我的這個請求被拒絕了，但是不等於我這個人被他拒絕了，他拒絕的是我這個行爲。

第四，就是很多人會認爲，跟其他人提出需求代表着我們是很脆弱。實際上，親密關係就是一個互幫互助，一起成長的過程。所以說，親密關係交流法則的第一步就是表達你的需求。原則聽起來很雞湯：那就是敞開心扉，交流法則需要你真實的面對你的感受。

做諮詢的時候經常遇見女生跟我這麼講我心裏是這樣想的，但是我是那樣說的，對方應該知道我是怎麼想的，遇見這樣的問題我會問：對方有沒有讀心術呢？如果沒有，對方怎麼可能知道你在想些什麼呢？

親密關係中的正向表達

當我們要了解一個車的時候，我們需要打開車的零件。然後要知道哪裏是加油的怎麼保養的，一些基本的維修常識。對於人類也是這樣的。人類雖然千差萬別，我們有不同的膚色，吃不同的食物，在不同的文化下成長，但是人類還是有一些基本的共性的。在交流方面，我們很渴望三件事情：被理解，被感激，被接納。其實這三方面，也是我們在人生這條路上奔馳的汽油。他們也是我們愛別人和被別人愛的基本要素，缺乏這些，我們會變得很強的防禦性，不讓任何人進入我們的世界。

1、被理解指的就是同理心，我們每個人身上都攜帶者防禦警報，當任何人試圖指責我們的時候，我們的警報就會響起。他會自動關閉我們的交流和聆聽系統，這樣無論對方說什麼，都是在浪費時間。同理心，是指你能接受你的伴侶可以用他們的方式去感受這個世界，並不等同於去同意別人的觀點和做法。這隻能說你接納了別人用他們的方式去看待問題。

比如小紅跟小明說：“你去把地掃一下。”
小明說：“別命令我！”
小紅說：“我哪命令你了？你有病吧”

其實同理心就是小紅不一定認爲她真的命令了小明 但是她不會去攻擊小明的感受，她允許小明對這個事情有不同的感受，無論他是對的還是錯的。

2、感恩聽起來是跟同理心不同的詞彙 我理解的感恩是你告訴你的伴侶你多麼感激和在乎對方。對於感恩，很多人明明喜歡對方身上很多東西，都憋住不說，覺得會讓自己卑微，有的還喜歡說出來的都是對方的缺點，其實適當的感恩表達這會幫助你建立自尊。

更重要的，當你們之間存在困難的時候，表達感激可以讓伴侶去減少他的防禦系統。這樣他會更多的去傾聽你。可以每個月定期找一個時間，去進行感恩表達，你更多的表達你的感恩會幫助你用這個方法更好的。

比如說：我很爲你感到驕傲！當你幫助做輪椅的老人，你總是做些很小的事情，比如送花或者給我留愛的小字條，這些都讓我很感激。

大家即使不是當事人，聽見剛纔這些話 也能感覺到愛意了吧。我們也可以嘗試着對我們親密關係的人去表達這些。在工作坊的時候，我們有時候會玩玩偶遊戲，Wonderful 夫妻，當 Wonderful 先生說謝謝你今天讓我感覺如此美好，太太會說是的我也覺得跟你在一起的時光很美妙。說完之後家就開始笑。因爲大家表示在日常生活裏，幾乎從來不會對另一半說這樣的話。但是我想問問大家，誰不願意去聽到這樣的話呢。可是大家表達這些話的時候，爲什麼要通過玩具呢爲什麼我們不能夠告訴伴侶我們多麼在乎對方呢？

3、接納意味着，你愛對方，因爲她是他自己。好的，壞的，所有的。我發現很多伴侶都很害怕去接納他的伴侶，因爲他們擔心如果無條件的去接納伴侶，這會讓他們對他們越來越差。但是研究表明相反，當人們被全部的接納，人們會努力的讓對方滿意，當然無條件的接納伴侶是很難的事情。

很多時候我們都認爲，只要我們伴侶更耐心，更有錢，更帥氣等等，我們才能去接納她們。很多人跟伴侶在一起時間長了，基本上就忽視了對方的優點，就像我們是不完美的一樣，我們的伴侶也是不完美的，我們伴侶有時候也會做出來一些錯誤的事情。我們可以問問自己，也許他的某些行爲是做的不對的，但是也許動機是好的。

接納伴侶需要一些小練習，比如：她這麼做的背後是有什麼積極的動機？當你意識到你的伴侶是一個人類，有時候會有情緒上的痛苦，那有時候用了一些攻擊的原因去讓你分擔情緒。就想當我們是嬰兒的時候，我們會通過哭泣是表達我們的感受，可是當我們是成年人，我們會掩蓋我們的脆弱，我們可能用更強勢和攻擊性的行爲去表達脆弱。接納意味着不困惑，意味着你總是會愛她，即使她有時候並不那麼完美。

接納也意味着我們同意伴侶跟我們會有許多的不同，比如說小明的媽媽是個喜歡說：我們父母這麼做，還是不因爲我們愛你，在乎你。所以小明的潛意識裏，愛這個詞就伴隨着壓力和不舒服。所以小明在感情裏很少在嘴上表達愛意，這可能對小紅來說就是一沒有安全感的事情。但是如果小紅能夠理解並且接納小明的感受和經歷，也許兩個人會少掉很多爭執。

我們可以問你如果給你五百塊錢，讓你在一分鐘之內，給你的伴侶惹生氣。很多人都表示能做到？我們反過來問，如果給你五百塊錢，讓你一分鐘之內，讓你的伴侶感到被愛，大家有什麼辦法嗎？可能很多人都答不上來，有的人可能覺得跟另一半培養感情是件很難的事情。

其實不是這樣的，科學研究表明，如果我們在日常可以理解，感恩，接納對方，增加我們情感的濃度，這就相當於每天都給愛情銀行裏存錢。在愛情銀行裏存錢的方式裏，除了說感恩這種語言上的表達，還有非語言的表達，比如說肢體語言交流和麪部語言交流。我經常聽見有的女生跟我說，我生氣的不是這件事，而是他那個態度。可見交流中的態度是很重要的。研究表示，僅僅是在監獄裏面的牆上掛上一個笑臉，就可以降低百分之二十五的暴力事件。當我們臭着臉面對伴侶的時候，她們本能的會感覺到自己被攻擊或者被嫌棄，不如我們去經常去對我們的伴侶微笑，擁抱我們的伴侶，讓他們感受到我們的愛。

如何處理親密關係中的衝突？

講完親密關係的正向的表達，我們來聊聊我們如何處理親密關係中的衝突，這個可能是大家比較關注的問題。

交流法則#1：當我在交流的時候，我有沒有聆聽對方。

我經常也會聽到，大家覺得對伴侶提意見很難，我去問你們是怎麼交流的，她說我怎麼樣怎麼樣，我說你這個交流是單方面的。你有沒有知道對方在想什麼，給對方表達感受的機會呢？

當我們去解決問題，我們想象一下去面對一個三歲孩子的感受。當你做了一些傷害他的行爲，他可能會哭泣，那你會怎麼做呢？你可能不會繼續再指責他，相反，你可能會有很多的憐惜和疼愛。這時候等他平靜下來，你可能會給他提供一些建議。怎麼可以在未來處理的更好。當我們生氣的時候，其實我們就像三歲的孩子一樣脆弱，我們需要別人知道我們是感覺多麼糟糕。在我們的生活中，大部分的人都不會去同意別人，大家更願意去“糾正”別人。當其他人去表達他的痛苦，馬上拒絕對方的言論，當我們感受到痛苦，我們第一需要我們的感受被理解，之後我們才願意去修復和解決問題。

交流法則#2：當我在交流的時候，我有沒有指責對方？

衝突總會出現在兩個人之間，但是衝突本身是沒有錯的，事實上親密關係中總是會有衝突存在的。重點是我們更有效的去解決衝突。有一個比衝突更可怕的東西，存在於解決問題的交流中。那就是指責。很多人常常喜歡攻擊別人，和指責別人的錯誤，來證明自己是對的。很多人認爲畢竟我這件事全都是對的。他都是錯的，不可理喻的。很多人認爲他們在交流，其實他們在攻擊，在指責，很多人說我沒有喊，我在心平氣和跟她說，其實交流的內容就是不斷逼對方認錯。我們的自尊心可能就像個銀行一樣，我們打比方說每個人的自尊心銀行都有十塊錢，那麼當一個人的自尊心銀行只有兩塊錢的時候，他們就可能有暴力行爲。在新聞上，我們可以看到有的人，僅僅因爲在路上多看了其他人幾眼，就捱了打。那些因爲被人看了就覺得有攻擊型的人，可能自尊銀行裏只有兩塊錢。而那些自尊銀行只有 0 塊錢的人，他們常常會折磨死自己。

指責總會消耗我們的銀行財產。當你指責你的伴侶的時候，你的伴侶會感受到自己被攻擊。這時候他會爲自己的自尊做出辯解。

比如，小紅說：“你很自私！”
小明說：“你認爲我很自私？你該照照鏡子，你纔是最自私的那個！”

大家都在電影裏看過這樣的場景，大家都覺得這樣的對話很傷人並且沒什麼意義。沒有人會從這樣的對話中學會愛和尊重。如果你的對象已經很不開心了，如果你希望對方聆聽你，那麼就要給他足夠的理解，感恩和接納。這些就像在你的伴侶的銀行裏存了錢一樣，當他的自尊銀行的存款上升。他會自然的去聆聽你，關心你。 同樣的，如果你的伴侶感受到很大的壓力。那麼最好的事情，是去他的自尊銀行存錢。這樣他感受到你對她的愛，他也會愛你，會行成一個正向循環。

交流法則#3：當我在交流的時候，我有沒有把一個人的行爲上升到一個人的品質。

這樣的情況也時常發生在我們的生活場景裏，比如很多人都經歷過這樣的情況，我們小時候媽媽讓你去寫作業。

媽媽說：“小紅，快去寫作業。”
小紅說：“一會兒去。”
媽媽說：“小紅，快寫作業，怎麼總是這樣呢？”
小紅沒有動，這時候媽媽發脾氣了：“說你這個孩子就是懶！沒救了！”

這個是很典型的把一個人的行爲變成攻擊她的習慣，最後變成攻擊這個人的品質，我們的感受肯定是不好的。跟伴侶也是這樣，即使對方有我們不喜歡的行爲，我們不要上升到攻擊對方的品質。

交流法則#4：交流的時候我有沒有同理心。

在我們要求伴侶聆聽我們之前，我們要先去理解下伴侶的感受，即使有的聽起來並不合理。

關於同理心有幾個表達式：

這聽起來，你好像很難過 / 沮喪……

這一定會，感覺很失望 / 難過 / 沮喪……

我很抱歉，讓你感到失望 / 難過 / 沮喪……

交流法則#5：我們要問問自己，是對重要還是被愛重要？

首先要承認親密關係中，有很多個人原則，廣義上說親密關係中的暴力和虐待行爲，不論是身體上，還是情感上，都不提倡留在這段關係裏。對於每個人，大家的原則都不一樣，有的人覺得家人是他的底線，有的人覺得尊嚴是他的底線。除了這些原則性問題，我們在處理親密關係中的衝突的時候。我們可以要做一些讓步。拋開原則性問題時候，我們會問自己如果我堅持自己是對的，會發生什麼，比如小紅堅持自己是對的，小明是錯的，說小明很自私。那麼小紅的期望是什麼呢?小紅可能一方面站在了感情的制高點，獲得了尊嚴，一方面潛意識是希望通過指責讓小明改變。但是現實事情呢?可能是小紅說小明自私，小明反過來攻擊小紅說，你更自私。

每個人都是有自由意志的。大部分時候，當我們去指責對方，對方也會指責回來，然後互相傷害，進入惡性循環。關於親密關係的交流，有一個壞消息。那就是你如果想要一段快樂和健康的關係，你可能要放棄一些事情和觀點，如果你總是想堅持自己是對的，並且一味指責對方，同時還想要有一段健康的親密關係的話那是不可能的，這就像一個屋子裏絕對的黑暗和光明是同時存在的。

交流之前的準備：有的人會問我，你鼓勵我會表達自己的感受，難道我每件事情都要拿出來說嗎？改變是很難的。想象一下另外一個人要求你去改變。這會讓你覺得自己很糟糕。所以如果你想要你的伴侶改變他的行爲，我們要小心去提出自己要求。在我們提出我們的需求，交流之前，我們可以問問自己這些問題：

1.爲什麼我會覺得沒有安全感？

2.對方哪些具體的行爲讓我感覺到這些？

3.對方哪些具體的行爲讓我感到愛和安全感？

4.哪些事情和行爲我想拿出來討論？

5.這些要討論的行爲 都有哪些可以處理的辦法？

交流法則#6，具體化你的需求。

我們在開頭說有人更喜歡去跟計算機對話，因爲計算機有明確的指令和要求。很多時候當我們跟伴侶表達我們的需求的時候，伴侶並沒不知道怎麼去做。比如小紅說我很不舒服。小紅期待小明能理解馬上給他送吃的。可能小明說多喝熱水，小紅就很生氣。所以最好要明確表達你的需求。總結起來，就是當伴侶比較生氣的時候，我們可能要說：

這聽起來，你好像很難過 / 沮喪……

這一定會，感覺很失望 / 難過 / 沮喪……

我很抱歉，讓你感到失望 / 難過 / 沮喪……

爲什麼你會有這種感受？

我可以幫助你做些什麼呢？

比如小紅對小明說：“我很抱歉讓你感覺自己被命令了，是我說話的語氣還是什麼讓你這麼感覺的呢？我怎麼樣下次能讓你不會有這樣的感覺呢？”
當我們有需求的時候，我們可以嘗試這樣的表達：“你的 XX 行爲，我感到非常 XX。我們可以下次換個方式或者辦法嗎？”比如“你不去拖地這個行爲，屋子髒我感覺到很焦慮，下次可以回家之後馬上就拖地嗎？”

有時候我們會說，即使這樣，我們用了這樣的辦法，交流還是無效的，沒有辦法的。首先我們要承認親密關係中存在問題是很正常的。交流不一定能解決所有的問題，這就像跑步有利於我們身體健康，但是跑步不能保證身體一定不會生病。如果交流之後還是沒有解決問題，那我們去了解一下對方的需要，你可以試着溝通說：“上次你好像已經同意了什麼什麼，但是看起來沒有發生，你覺得有什麼可以改善的辦法嗎？”記得這時候 要尊重伴侶的意見，看看有沒有可能尋找一個更好的解決辦法。

好好說話有時候就像學一門外語，即使我們今天瞭解知識，我們應用到生活裏面，還需要一些練習和時間。最後，希望每個人都能收穫一份健康、美好的親密關係！
