一無所長的情況下怎麼找工作？

關於找工作、跳槽、薪水、上升空間等，這都是永恆的話題。在此，我用我十多年的經驗，總結了如下七條找工作的方法：

1. 優先做自己感興趣的工作

以前我講關於職業的話題，收到了很多人的郵件，有一位 Y 同學讓我感觸特別深。

他 3 年前畢業，稀裏糊塗地進入互聯網做了產品經理，但其實他對產品經理，對所屬的行業和產品並不是那麼感興趣。總歸做了幾年了，月薪也一萬多，但就是睡不着，因爲他認爲自己在邏輯、流程方面不強，特別怕過需求這個環節，繼而對自己產生了懷疑。但現在又沒辦法去做自己感興趣的事，因爲有老婆、孩子，房子還有 27 年的月供未還，問我該怎麼辦。

我能理解他的那種無力感，因爲類似的同學我接觸過很多，比如有些在 BAT 大公司裏，做着不那麼感興趣的事情；比如文科生去做商業產品，在數據部門做產品設計，很吃力卻又走不了，大公司安逸的環境和不錯的薪資都是誘人的主要因素，最後導致了溫水煮青蛙。

十多年前我工作的時候，作爲學渣，我能找到一份工作就可以了，根本沒機會去做所謂的職業規劃。所以那個年代，“幹一行愛一行”是讓自己活得沒那麼累的方法，因此我建議 Y 同學也是幹一行愛一行吧。

不過現在的年輕人，剛畢業或者畢業 2、3 年內的，選擇就多了，畢竟 90 後的同學，養家餬口的壓力也沒那麼大了。在這種情況下，我是特別建議，在畢業之前多嘗試不同的工作，去不同的行業實習，找到興趣點，“先愛一行再去幹一行”。

比如，我們的曉玲同學，之前做編輯，寫文案簡直要命了，後來有機會接觸了社羣運營，和人打交道是她擅長並且喜歡的，就會發自內心的喜歡這事，至於公司大小先往後放，做自己喜歡的事情最重要。

從思考的角度，換工作和找工作，都是從以下六個角度去組合的。

1. 名：職位高低；
2. 利：收入高低；
3. 興趣：所做的事情是否是自己感興趣的；
4. 領導：直屬領導的風格；
5. 團隊：所在的團隊的風格；
6. 公司：企業所處的區間，是成長期還是衰退期等。

對於畢業生、2 年內的同學，特別建議的選擇是，在一個不錯的團隊，做自己感興趣的事情，興趣是自驅力的唯一來源。

3 年 -5 年的同學，建議是在一個不錯的平臺，做感興趣的事情，因爲你之前的工作經驗，需要到更大的平臺去挑戰一下了。對於收入的部分，至少在職業早期，不應該成爲你關鍵的考量點。

2. 如果你沒有長期目標，那跟鹹魚有什麼區別？

面試的時候，很多面試官都喜歡問：你希望 5 年後的自己是什麼樣子的？這個問題是一個很容易冷場的問題，因爲沒想過，或者沒認真想過。

三節課招募的其中一條，就是必須說明未來 3-5 年的目標，並且說明已經爲這個目標做了哪些努力。因爲我們認爲，有清晰目標的人，對於學習的路徑選擇更清晰，對學習的投入程度也會更好。

關於目標，我們可以用以下的幾種自我提問方式：

1. 中長期的計劃與路徑是什麼？
2. 短期內需要迫切解決的問題是什麼？
3. 自身的優勢是什麼？和長期計劃的關聯度如何？
4. 現在離職的原因關鍵點是什麼？
5. 換工作能解決這個問題嗎？

如果自問完，回頭去看，覺得現在的工作和中長期目標關聯度不大，那就趕快果斷換，如果關聯度很大，但工資不如意，那我建議先把事情做好吧。

3. 如果你現在乾的不好，換一份工作也一定不會好

小 C 同學說，公司不靠譜、團隊也不靠譜，也沒人帶，我想換個領導好、公司文化好、團隊氛圍也好的團隊。

追問之下，發現小 C 同學現在做的事情，離專業還有很大距離。從工作的角度來看，需求文檔做完後，評審結束也不更新，項目推進過程中，準點下班，研發在加班。說白了就是既不專業也不職業，換一份工作，我估計也沒啥區別。

後來我給她的建議是：

1. 把之前做過的所有文檔全部收拾一遍，並放到公司內網；
2. 把最近的項目上線後的數據表現，全部導出來分析併產出報告，發給相關的負責人；
3. 把進行中的項目進度，每天記錄，每週發出來；
4. 上線郵件認真寫，而不是泛泛的“感謝研發同學的努力”，而是有數據對比、有功能清單、有具體感謝的人和點滴事項。

2 個月後，小 C 同學的反饋是，雖然自己的專業能力提升並不大，但是好像突然感覺團隊的人靠譜了，研發也願意和她交流了，運營的同學也更願意把用戶的問題交給她去處理了。

其實，環境沒變，是自己改變了，結果就已經在改變了。

4. 轉型意味着賭上一切，值得嗎？

D 同學，是一名 3 年的運營經理，想要轉成產品經理，於是開始參加各種分享會，也滲透到了 N 個產品經理的羣裏面，自己本身的運營工作的時間就變少了，公司內部評級是 C（中庸），其實他也不是沒有機會，但薪資、職位、公司等都不如意，1 年後的現在，還是做着一般般的運營，於是他想找一份牛逼的工作。

E 同學，是一名 4 年的研發，也想轉型做產品經理，一開始買了一些書、線上的課程學習。接着自己就在工作之餘，搭建了一個小小的興趣社區，做功能的時候，自己也寫文檔，做流程和原型，然後自己寫代碼，幾個月後收穫了幾千用戶。

他覺得差不多了，就裸辭，給自己定了 2 個月找一份產品工作的目標，最後成爲了一個創業公司唯一的 PM。雖然薪資只有研發的一半，但是轉型成功後 2 年不到，現在已經是總監，並帶着一個 10 個人組成的產品團隊了。

這兩者原因是什麼？轉型是要堵上一切的，如果患得患失，機會就不會大。關於轉行這個事情，其實也是有一些問題可以用於自檢自查的：

1. 轉行的原因是什麼：工作不順？只是感興趣？覺得很酷？
2. 做了哪些努力？爲了轉換你正在、打算做什麼努力？
3. 可以和不可以放棄的東西：人脈、職位、薪資、團隊、積累等；
4. 可預期的結果：什麼時候開始，什麼時候結束？

據說，有 80% 的人問完自己以上問題後，默默搬磚去了。

5. 簡歷如臉，你敢不要臉，活該別人看不起你

我大概從 2007 年負責項目開始，就會面對一堆的簡歷，其中有超過一半的簡歷幾乎會被我直接進垃圾箱，太差了。簡歷真的跟臉面一樣，除非你已經有獵頭幫你服務了，不需要靠簡歷吃飯了，否則還是好好對待你的簡歷吧。

下面是之前收到過的一份簡歷，看完後我整個人是崩潰的。

其實簡歷吧，你可以很簡單，但也可以很認真，我總結下我認爲比較好的簡歷需要表現出來的一些內容：

1. 匹配度高：我招用戶運營，你簡歷重點突出用戶運營的崗位和經歷，在 HR 初篩的時候命中率當然高了；
2.個性化：儘可能爲一個公司、或者一個行業準備一份特別的簡歷，而不是一處水源供全球，一份簡歷投所有；
3.簡潔：基本簡歷不超過 2 頁紙，寧少勿多，有補充材料可以增加附件；
4.重點突出：列出二十個做過的項目，不如把一個最值得說的項目講清楚，有過程有結果；
5.查看方便：能用 PDF 就不要用 Word，不要壓縮包；
6.表達明瞭：直接投遞的郵件中，正文把要點寫明白，別直接一句“這是我的簡歷，請查收”就完事了。

對於簡歷，我認爲首先你要對用人單位的需求先調研清楚了再準備，這樣你的匹配度纔會很高。因爲同樣的崗位，不同公司的要求都不一樣。比如從下面的 JD 裏面，如果你是做 C 端產品的來投遞，那就要表現出你對類 B2B 電商平臺的瞭解程度，否則你的簡歷肯定會被扔掉。

6. 海投簡歷效率低，投簡歷也是一門學問

E 同學就瞄準了某獨角獸一後端產品的職位，投遞一次未果，然後就每天都投一次，拉勾上有限制了，就換一個馬甲帳號繼續投，頗有一種不成功便成仁的牛勁，實際上這樣的做法一點意義都沒有。

後來我建議他，你把簡歷準備好了，然後把目標公司分成 ABC 三層，每個公司的職位也分成一二三檔，投遞的時候，先投 C 層的一檔，練練手，看看投遞被約談的轉化率，去感受一下。然後去力保 B 層公司的三檔職位或者是 A 層公司的 C 檔職位，將其作爲你的基本保證，在這裏應該能找到一個保底的工作，最後再去挑戰 A 層公司的一檔職位。

比如，下圖是一個英語比較好，對旅遊、教育和跨境電商比較感興趣的產品同學所列出的對於目標職位的分解思路。

其實投簡歷和做產品是一樣的，需要從小範圍測試再到大批量上線，分層投遞就是測試、迭代和成長的過程。

7. 爲什麼找工作要用半年時間來準備？

F 同學非常用心，春節一過便開始找產品助理的相關工作，投簡歷和麪試時都會帶一份“調研報告”，以此證明對這家企業的瞭解。爲了找工作，短期內在知乎上回答問題，在簡書上發東西，不過在面試的時候，用人方的評價是“很功利，爲了找工作，臨時做了很多的準備”。

最後雖沒有跳槽成功，但是在知乎和簡書上寫東西的習慣並沒有丟棄，在去年年底，有人主動挖他，說他一直在寫的東西很有意思，覺得他觀察還挺深的，於是最後 F 同學跳槽成功了。

越着急，越混亂，反而是長期堅持的輸出，在未來一段時間能收穫更好的回報。因爲對於用人方來說，更希望看到的是一個人長期的思考，而不是短期的產出。

對於找工作來說，運營自己是很關鍵的，從運營的角度來看，短期刺激用戶的留存並不會太好，長期的跟蹤纔會帶來忠實的粉絲。

所以，想找工作就得提前三個月到半年的準備，這期間，想明白上面關於目標、興趣等方面的問題，然後就是運營自己，可以嘗試以下幾個方面。

對行業的跟蹤：

1. 從行業到產品，行業的運轉邏輯和主流產品；
2. 從產品到細節，關鍵玩家的制勝點是什麼；
3. 發現機會，並有條件去驗證機會。

對產品的跟蹤：

1. 從用戶到粉絲的轉變，深度使用並表現自己；
2. 從粉絲到達人，玩轉產品，成爲少數用戶，成爲產品的貢獻者；
3. 從達人到員工，加入團隊，改變產品。

我們三節課的產品、運營員工，大部分都是來自於在我們三節課學習的同學，道理也是如此，這些同學玩的越好，機會也越多。從公司的角度來說，當然會優先找熟悉自己產品的員工，其歸屬感更好，留下的的概率也會更大了。

說在最後，我覺得，找工作是非常慎重的事情，所有的思考都只是爲了讓自己不後悔。
