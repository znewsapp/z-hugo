蒸汽機車進入正線運轉，在技術上並不存在特別多的難題。電氣化區間可能會造成接觸網老化速度加快，但是並不會造成結露 / 掛冰。目前在正線上運營蒸汽機車比較多的國家有英國，南非，美國等。國內目前運行的蒸汽機車基本上都是屬於調車 / 支線小運轉，正線基本上已經沒有了，不過很多國鐵站點依然有讓蒸汽機車維持運轉的能力。

-

先說下國外的情況：

北美：

Class 2/3 級的小型鐵路公司保留蒸汽機車運轉的非常多，例如當年從集通鐵路購買 QJ 級的 Iowa Interstate (衣阿華州際) 等，Class 1 級的官方運轉目前只有 Union Pacific (聯合太平洋)。

圖 1. Union Pacific FEF-3 Class No. 844 (i)

英國：

英國鐵路私營化之後，部分難以盈利的 / 技術規格老化的區間被出售給私人變成遺產鐵路(Heitage Railway)，這些私營的線路里保留了不少當年英國國鐵退役的蒸汽機車，也有從國外買進來的，比如從中國買進來的 KD6。目前大部分進入正線的長途蒸汽機車牽引的運轉都是由大英鐵路博物館負責協調 / 維護的。

自從 1960 年起，英國鐵路系統不再引入蒸汽機車，但是 94 年一個由車迷發起的基金會歷時 14 年重新造了臺新的蒸汽機車（圖 2）。

圖 2. British Railways A1 Class 60163 "Tornado" (ii)

圖 3. British Railways Standard Class 7 70013 "Oliver Cromwell" (iii)

如果題主考慮的是讓蒸汽機車進入國鐵正線運轉的話，應當要先解決這幾個問題：

1. 線路標準（例如曲線半徑和限界）

2. 機車交路（如何保證機車的可用性和可靠性？如果一個比較長的交路的話調度可能是個問題）

3. 聯控設備 （機車信號，以及列尾裝置）

4. 機務設施（蒸汽機維修保養和內燃 / 電力沒有可比性）

-

1. 單獨討論限界的話，大部分區間都是可以對蒸汽機車比較友好的；然而因爲蒸汽機車排氣對於背壓要求比較高，長大隧道和低淨空區間對於長期性運轉並不合適。另外因爲機車走行部受車架限制大，某些新建機務段可能並不具有容納蒸汽機車的能力；蒸汽機車整備只能在側線進行，然而這存在一個未知的安全風險，以及效率也是個問題。

2. 機車交路一直是蒸汽機車的軟肋，但是客貨分離後蒸汽走既有線的調度問題可能還好解決，但是這時候續航又是個問題；國內基本上都是小運轉就和續航本身的關係非常大。英國 / 南非等是有既有設施未淘汰，北美的做法是加掛一節煤水車（實際上主要是裝水）。後者可能對於國內之後的正線蒸汽機車運轉更具有參考性。以及蒸汽機車本身不能經常熄爐，不然對於鍋爐壽命影響非常大，長期的燃料費用也是非常可觀的￥￥￥。

3. 聯控和列尾…只能指望甲方 push 乙方工程師了，理論上難度並不是很大，然而蒸汽機車制動機和內燃 / 電力不一樣：壓縮機做功來源是蒸汽，另外蒸汽司機本身的操作已經很多了，如何確保不添堵又是一個問題。

4. 主要的問題在於架修廠和轉向設備（人字線 / 回車盤），蒸汽機車理論上可以後退，但是在運轉中是非常避諱的一件事，因爲瞭望條件不夠；另外因爲結構本身的問題，蒸汽機車的完全檢修需要把機車吊起來（更換輪對之類的）。不少蒸汽機車都到了大修的年齡，以至於需要非常徹底的維修才能保證安全。到達壽命的部件需要被替換，以及性能劣化的部件需要補強，在缺乏大規模生產的情況下成本非常高。（尤其是輪對等鑄造件需要非常精密；另外機車的踏面製造難度也是非常高。）

另外 @懶洋洋 提到的滑動軸承的問題實際上是限於解放時期的工業生產能力，大部分機車使用滑動軸承是爲了避免過於複雜的維護；實際上滿鐵的部分機車（SL7/SL8 型）都是滾動軸承，在沈局一直用到 80 年代末期，現在被博物館收藏了。

其實維護的問題非常早就出現了，尤其是英國；爲了省人工和成本乾脆就讓一批機車乾脆進廠重建一次，所以有不少打着重建(Rebuilt)旗號，但是各個性能指標和舊型差距非常大的機車。

可能的實現方案

保留現有的機車隊伍可能是一個臨時的低成本解決方案，然而保養起來的困難程度很可能導致後續經營不了了之...因爲成本和人力都吃不消。

如果參照英國人的套路重建一批 QJ/JS 也是可以，一個是因爲很多機車構造年齡比較低（QJ/JS 都是 80 年代末期停產），另外一個是保有量非常大。但是爲了經濟性 / 環保法規要求方面考慮，應該重新改造鍋爐。具體的提升應該包括而不限於：燃燒方式（波特爐改造在 QJ 上進行過試驗，但是沒有投入後續生產），鍋爐壓力，過熱器，乏汽裝置等...這樣一個是可以降低運營成本，另外也可以針對目前或者未來的鐵路系統進行某些功能上的預留。另外煤水車應該優先考慮設計成兩節以上來維持比較長的續航距離。

圖 6. 運轉中的 Union Pacific FEF-3 Class No. 844，注意煤水車後面附加的 6 軸水車。即使原配水車可以裝載 89 噸水，但是在乾旱缺水的大盆地地區依然不是足夠可靠的。

凝汽式煤水車也可以考慮，但是某些情況下可能會損失牽引效率。凝汽式設計也可能帶來某些不必要的複雜裝置，滿鐵在 41 年曾經對機車進行過凝汽式改造，實驗續航能達到 1600 公里，但是並沒有繼續量產。目前還在少量使用的只剩下南非鐵路 25 型凝汽式蒸汽機車 /SAR Class 25。

但是實際上考慮到和現有的使用場景對應的話，重新測繪或者使用 RM/SL 7/SL 8/KF 型的圖紙進行建造可能是更合適的方案。重載電氣化鐵路的下基本上很難用蒸汽機車牽引貨運列車，國內以前已有的幾種客運 / 混合型機車可以具有更高的運行速度（110-130 km/h），改造鍋爐等結構件，達到可靠性要求之後，應該可以適應既有線的運行 - 調度機制。

暫時的結論

如果能把蒸汽機車引入正線牽引的話，應該在達到整備 / 維修條件下，同時具有修理廠。但是考慮到蒸汽機車存量少，維修大的糟糕現狀，修理廠基本上也應該具有製造蒸汽機車的能力。遠期的話應該參照和借鑑 60-80 年代國外最後一代設計的經驗和教訓和國內的運轉經驗重新做一批圖紙，在可能的條件下維持小批量，但是可持續性的幾種型號的生產。
